# Métodos de Interpolação e Aproximação
Projeto desenvolvido para aplicar métodos de interpolação e aproximação de funções para a disciplina de Cálculo Numérico Computacional na UFPEL.

- Interpolação de Lagrange
    - Constrói um polinômio interpolador único que passa pelos pontos dados

- Interpolação de Newton
    - Constrói o polinômio interpolador usando diferenças divididas, permite reaproveitar cálculo para os outros pontos, então para um conjunto grande dados é eficiente

- Interpolação por Splines Cúbicas
    - Divide o intervalo em subintervalos e ajusta cada parte em polinômios cúbicos

- Aproximação Linear(Mínimos Quadrados)
    - Ajusta uma reta aos dados minimizando o erro quadrático
