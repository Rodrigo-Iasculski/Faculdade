# Métodos para encontrar raizes
Projeto feito para encontrar raizes de sistemas de equações lineares para a matéria de Cálculo Numérico Computacional na UFPEL.
Métodos:
- Bissecção
  - Repetidamente divide o intervalo ao meio e mantém o subintervalo onde ocorre a mudança de sinal.
- Falsa Posição
  - Em vez de dividir o intervalo ao meio, ele calcula a interseção da linha que passa pelos pontos (a,f(a)) e (b,f(b)) com o eixo x
- Newton Raphson
  - Utiliza a derivada para fazer a aproximação
- Secante
    - Uma melhora do Newton Raphson sem utilizar a derivada
